# api-node-aula-fmu

## Projeto de API com Node.js

### Instalação
Passo 1: Clone o repositório
```bash 
git clone https://github.com/hsouzaeduardo/api-node-aula-fmu.git

```

Passo 2: Instale as dependências
```bash
npm install

```

Passo 3: Inicie o servidor
```bash
node app.js

```

### Rotas
# [GET] /api/:id/estados
# [GET] /api/:id/estados/csv
# [GET] /api/estados/csv
